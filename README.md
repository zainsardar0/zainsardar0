### ☁️ Weather MCP Server
**Repo:** [`weather-mcp-server`](https://github.com/zainsardar0/weather-mcp-server)

Built a **production-ready MCP (Model Context Protocol) server** for real-time weather data powered by the Open-Meteo API.
This server enables AI agents to query live weather conditions, making it a practical tool for LLM-based applications.

**Key Features:**
- Real-time weather data via Open-Meteo API
- Full MCP protocol compliance
- Production-ready TypeScript implementation
- AI agent integration ready

**📌 Outcome:** A fully functional MCP server that can be plugged into any LLM application for live weather awareness.

**Tech Used:**
